get(self, keys)

source code 

An internal method used for locating specific objects within the feed. It assumes the key list to contain the path to the object of interest.

Parameters:
  • keys - A list of the keys along the path to the object of interst. Note, if the keylist goes through a list object, the index is used as the key.
Returns:
The object or raise an exception if the path is invalid.

get_entries(self)

source code 

This assumes a top level 'entries' entity, which can be expected for RSS news feeds. This method will return a dictionary keyed by the title of each story provided. Since reply messages tend to share title, we use a list to represent the links.
